# Commands

| Action                | Command                                            |
| --------------------- | -------------------------------------------------- |
| Add all               | git add .                                          |
| Creating a new branch | git checkout -b branchName                         |
| Switching branch      | git checkout branchName                            |
| Rebasing              | git checkout main, git rebase branchName, git push |
| Stashing              | git Stash                                          |
